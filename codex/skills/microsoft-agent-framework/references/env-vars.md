# Environment Variables

Source: https://github.com/microsoft/agent-framework/blob/main/python/packages/core/README.md

Set these in your shell or in a `.env` file as required by the client you use:

```bash
OPENAI_API_KEY=...
OPENAI_CHAT_MODEL_ID=...
OPENAI_RESPONSES_MODEL_ID=...

AZURE_OPENAI_API_KEY=...
AZURE_OPENAI_ENDPOINT=...
AZURE_OPENAI_CHAT_DEPLOYMENT_NAME=...
AZURE_OPENAI_RESPONSES_DEPLOYMENT_NAME=...
AZURE_OPENAI_API_VERSION=...

AZURE_AI_PROJECT_ENDPOINT=...
AZURE_AI_MODEL_DEPLOYMENT_NAME=...
```

## Copilot Studio prerequisites

Source: https://github.com/microsoft/agent-framework/blob/main/python/samples/getting_started/agents/copilotstudio/README.md

```bash
COPILOTSTUDIOAGENT__ENVIRONMENTID=...
COPILOTSTUDIOAGENT__SCHEMANAME=...
COPILOTSTUDIOAGENT__AGENTAPPID=...
COPILOTSTUDIOAGENT__TENANTID=...
```

Notes:
- Variables vary by provider and client. Confirm required names for your scenario.
